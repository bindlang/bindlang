# Installation

[Back to Reference](index.md)

---

## Requirements

- **Python:** 3.10 or higher
- **Dependencies:** Pydantic 2.0+ (installed automatically)

---

## Install from PyPI

```bash
pip install bindlang
```

---

## Install from Source

```bash
pip install git+https://github.com/bindlang/bindlang.git
```

---

## Development Setup

```bash
git clone https://github.com/bindlang/bindlang.git
cd bindlang
pip install -e ".[dev]"
```

---

## Verification

```bash
python -c "import bindlang; print(f'bindlang {bindlang.__version__} installed')"
```
